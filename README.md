## Requirements

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/cyberwamods/bot-follow`

`$ cd bot-follow`

`$ chmod +x insfollow.sh`

`$ termux-wake-lock`

`$ bash insfollow.sh`
